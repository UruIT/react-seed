UruIT seed project for ReactJS applications


## Stack

* Webpack
* ES6/7
* ESLint
* React
* Fetch
* React-Router
* Jest
* Sass
* Express
* Git hooks
* Plop


## Secondary features

* Redux
* Redux-Segment
* MongoDB
* MSSQL
* i18n
* PDF

## Development

Restore all packages and start development server:

```bash
yarn
yarn dev
```

Open browser on [localhost:3000](http://localhost:3000/)
